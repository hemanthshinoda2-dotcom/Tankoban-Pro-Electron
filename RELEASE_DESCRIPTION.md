## Tankoban Pro

A powerful Windows desktop application for managing and reading manga/comic archives and video libraries.

### What's New
- **Enhanced Video Performance** — Improved playback and rendering for smoother video viewing
- **Context Menu Integration** — Right-click support for quick access to common actions
- **Chapter Management** — Better organization and navigation for multi-chapter content
- **Stability Improvements** — Bug fixes and performance optimizations

### Installation

**Option 1: Setup Installer (Recommended)**
- File: `Tankoban Pro-Setup.exe`
- Full installation with Start Menu shortcuts and file associations
- Automatically handles updates and dependencies

**Option 2: Portable Version**
- File: `Tankoban Pro-Portable.exe`
- No installation required — run directly from any folder
- Perfect for USB drives or isolated environments
- All settings and data stored in the app folder

### Key Features

- **Comic Archive Support** — Read CBZ and CBR files with smooth page navigation
- **Video Library Management** — Organize and play video content with MPV integration
- **Chapter Tracking** — Keep track of your reading progress across series
- **Modern Interface** — Clean, intuitive UI built with Electron
- **Library Scanning** — Automatic detection and organization of your content
- **Thumbnail Generation** — Quick visual browsing of your collection

### System Requirements

- OS: Windows 10 or Windows 11 (64-bit)
- RAM: 4GB minimum, 8GB recommended
- Storage: 200MB for the app + space for your libraries
- Display: 1280x720 minimum resolution

### First Run

1. Launch Tankoban Pro
2. Set up your library folders for manga/comics and videos
3. Let the app scan and index your content
4. Start reading and watching!

### Known Issues

- macOS and Linux are not currently supported
- First library scan may take time depending on collection size
