# User Release Guide (Windows)

This guide is for end users who want to run Tankoban Pro Electron without building from source.

## Quick start

1. Download the latest Windows release asset from the project Releases page.
2. Extract it to a normal user-writable folder.
3. Launch the packaged executable.

## Support scope

- Windows only (Windows 10 and 11)
- Tested primarily on Windows 11

## Known limitations

- Linux and macOS are not supported.
- Source-build or debug steps are intentionally not included here.

If you need source setup, see [CONTRIBUTING.md](../CONTRIBUTING.md).
