# thumbs domain (app/main/domains/thumbs)

This domain stores and serves images used by the user interface.

It is used for:
- comic series thumbnails
- video show posters (custom pasted posters and auto posters)

## Key file

- `index.js`

## Persistence (user data)

- `thumbs/` folder and metadata files under Electron user data
- `video_posters/` folder for show posters (when used)
